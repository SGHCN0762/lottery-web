# 登录注册功能使用指南

## 🚀 快速开始

### 1. 启动开发服务器
```bash
npm run dev
```

### 2. 访问页面
- **登录页面**: http://localhost:5173/auth/login
- **注册页面**: http://localhost:5173/auth/register

## 💾 本地存储说明

登录成功后，以下数据会保存到 localStorage：

```javascript
// Token
localStorage.getItem('token') 
// 示例: "mock_token_1234567890"

// 用户信息
localStorage.getItem('userInfo')
// 示例: {"id":"10086","name":"娱乐达人","phone":"13800138000",...}

// 记住的手机号（如果勾选了"记住我"）
localStorage.getItem('rememberedPhone')
// 示例: "13800138000"
```

## 🐛 常见问题排查

### 1. 页面空白
- 检查控制台是否有错误
- 确认 Vite 开发服务器正常运行
- 检查路由配置是否正确

### 2. 样式异常
- 确认 Vant CSS 已引入
- 检查 CSS 变量是否定义
- 确认 Less 编译器正常工作

### 3. 路由跳转失败
- 检查路由名称是否正确
- 确认组件路径是否正确
- 查看控制台是否有路由错误

### 4. 表单验证不生效
- 检查 `:rules` 属性是否正确绑定
- 确认 `van-form` 组件正确使用
- 查看控制台是否有 Vue 警告
