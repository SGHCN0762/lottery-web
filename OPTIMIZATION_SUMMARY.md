## 🎨 设计规范

### 命名规范
- **CSS 类名**：kebab-case（如 `.hero-section`）
- **变量名**：camelCase（如 `themeMode`）
- **常量**：UPPER_SNAKE_CASE（如 `STORAGE_KEY`）

### 注释规范
- **文件头部**：说明文件职责
- **函数注释**：JSDoc 格式，包含参数和返回值
- **区块注释**：分隔不同功能区域
- **行内注释**：解释复杂逻辑

### 代码组织
```
组件结构：
├── template    // 模板
├── script      // 逻辑
│   ├── imports
│   ├── props
│   ├── 组合式API
│   ├── 计算属性
│   ├── 监听器
│   ├── 工具函数
│   └── 事件处理
└── style       // 样式
    ├── 主要样式
    ├── 响应式
    └── 动画
```

## 🔧 技术亮点

### 1. 设计令牌（Design Tokens）

### 2. 响应式设计
- 移动端优先
- 平板和桌面端适配

### 3. 无障碍访问（a11y）
- ARIA 属性
- 语义化标签
- 键盘导航支持

### 4. 性能优化
- 组件懒加载
- Keep-alive 缓存
- CSS 过渡替代 JS 动画

### 5. 用户体验
- 平滑滚动（iOS）
- 点击反馈动画
- 主题即时切换
